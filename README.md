## Required 
- Laravel 11
- Node v.19^


## Dependency
- Laravel 11
- Filament v.3
- TailwindCSS + DaisyUI
- Laravel Spatie

## Route
- `/admin` : untuk akses halaman admin filament

## Instalasi project

```base
> composer install
> php artisan migrate --seed
> npm install
> npm run dev
> php artisan serve
```
